# VanityParser

  parse text files from vanitygen into pretty JSON

  [![Build Status][travis-image]][travis-url]

Install
-------
stable
```bash
npm install -g vanity-parser
```
edge
```bash
npm install -g https://github.com/daxxog/vanity-parser/tarball/master
```
